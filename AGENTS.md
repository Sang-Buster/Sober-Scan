# AGENTS.md

Conventions for AI agents working on Sober-Scan, a Python package and CLI that
detects alcohol intoxication and drowsiness from facial images.

## Agent skills

### Issue tracker

Issues live on GitHub at `Sang-Buster/Sober-Scan` via the `gh` CLI. External
PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical defaults: `needs-triage`, `needs-info`, `ready-for-agent`,
`ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` and `docs/adr/` live at the repo root. See
`docs/agents/domain.md`.
